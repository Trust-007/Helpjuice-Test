> Describe your project in 1 or 2 sentences.

**[Helpjuice-test]** is a an app that mimics a rich text editor. It allows users to type in text and turn it into a heading using '/' + '1' and then clicking the option or hitting enter. You can also revert to a normal sized text using the same syntax.
